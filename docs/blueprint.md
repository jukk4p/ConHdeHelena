# **App Name**: Crafted Gifts

## Core Features:

- Animated Navigation: Animated Navbar: Smooth animations on hover and mobile menu transitions for enhanced user experience.
- Artisan Showcase: Homepage Banner: Features artisan-crafted gifts with subtle hints of professional tools like Cricut.
- Service Display: Services Catalog: Displays personalized offerings such as custom mugs, apparel, invitations, and vinyl decor.
- Project Insights: Blog/News Section: Provides gift-giving advice, trends, and project showcases with creation processes.
- Contact & Inquiries: Contact Form: Includes fields for name, email, message, special order inquiries, and a location map.
- Device Adaptability: Responsive Design: Ensures optimal viewing across all devices.

## Style Guidelines:

- Primary color: Muted rose (#E0B1CB) to evoke creativity and personalization.
- Background color: Light cream (#F5F5DC) for a soft, welcoming feel.
- Accent color: Pale gold (#DDD0C0) to highlight important details and CTAs.
- Headline font: 'Belleza', a humanist sans-serif, will give a personal touch to headlines.
- Body font: 'Alegreya', a humanist serif, for a pleasant reading experience in longer texts.
- Use FontAwesome or SVG icons to complement the content. Icons related to gifts, crafting, and personalization should be used
- Use subtle animations on scrolling to highlight content.
